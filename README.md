Getting Started

Proyek ini adalah titik awal untuk aplikasi Flutter yang dibuat untuk memenuhi tugas Ujian Tengah Semester (UTS) mata kuliah Pemrograman Seluler.

Aplikasi ini adalah purwarupa (mockup) sistem akademik sekolah sederhana yang memiliki fitur utama:

1. Halaman Login: Dilengkapi dengan validasi form (pengecekan format email dan kombinasi password).
2. Halaman Lupa Password: Simulasi pengiriman tautan reset password ke email.
3. Halaman Dashboard Akademik: Menampilkan menu akademik menggunakan GridView dan ListView.builder.

Lab: Write your first Flutter app
Cookbook: Useful Flutter samples
