# UTS Mobile Programming
Aplikasi Sistem Akademik Sekolah.

## Getting Started
Proyek ini adalah aplikasi Flutter yang dibuat untuk memenuhi tugas Ujian Tengah Semester (UTS) mata kuliah Mobile Programming.

Aplikasi ini merupakan sistem akademik sekolah sederhana yang memiliki fitur utama:

- Halaman Login dengan validasi email dan password.
- Halaman Lupa Password untuk simulasi reset password.
- Halaman Dashboard Akademik untuk menampilkan menu nilai, jadwal pelajaran, presensi, tugas, dan pengumuman.
- Menggunakan GridView dan ListView.builder untuk menampilkan data secara efisien.

A few resources to get you started if this is your first Flutter project:

- Lab: Write your first Flutter app
- Cookbook: Useful Flutter samples

For help getting started with Flutter development, view the online documentation, which offers tutorials, samples, guidance on mobile development, and a full API reference.
